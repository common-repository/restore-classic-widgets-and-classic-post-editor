== Description ==

Restore the previous WordPress classic widgets and post editor settings screens and disables the Gutenberg block editor from managing widgets or post editor.

There are cases where the our plugin will be necessary:
* Users simply preferring the old system.

Our plugin can restore the classic widget also post edit screen with just one click.
Turn the classic widget or post editor on or off from the settings page.
